API Endpoints

Authentication

POST /api/token/

Get JWT tokens

POST /api/token/refresh/

Refresh JWT token

Security

GET /api/security/2fa/setup/

Setup Two-Factor Authentication

GET /api/security/2fa/success/

2FA Setup Success

Active Workouts

GET /api/workouts/

List active workouts

POST /api/workouts/start/

Start workout session

POST /api/workouts/{id}/complete/

Complete workout

POST /api/workouts/{id}/exercises/{exercise_id}/

Log exercise progress

Exercises

GET /api/exercises/

List all exercises

POST /api/exercises/

Create new exercise

GET /api/exercises/{id}/

Get exercise details

PUT /api/exercises/{id}/

Update exercise

Workout Plans

GET /api/workout-plans/

List workout plans

POST /api/workout-plans/

Create workout plan

GET /api/workout-plans/{id}/sessions/

List plan sessions

Progress & Goals

GET /api/progress/tracking/

View progress history

POST /api/progress/tracking/

Log progress entry

GET /api/progress/goals/

List all goals

POST /api/progress/goals/

Create new goal

GET /api/progress/goals/{id}/

Get goal details

PUT /api/progress/goals/{id}/

Update goal

Built with Django REST Framework, PostgreSQL, and JWT Authentication
